# Gym_Master
A comprehensive Gym Management System built with C# and WPF, following Object-Oriented Programming (OOP) principles. This desktop application manages members, trainers, and subscriptions with a focus on clean architecture and in-memory data management.

  -- missons --
Models (Person 1): Create Member and Trainer classes (The Foundation).

Views (Person 2): Login Page design and authentication logic (The Gatekeeper).

Views (Person 3): Add Member UI and adding to the shared list.

Logic (Person 4): Design and functionalize Subscription Buttons and calculate dates .

Views (Person 5): Trainer Management UI (Add/Edit trainers).

Views (Person 6): DataGrid display and search/filter functionality.

Services (Person 7): Data Persistence (Saving/Loading to Text files).

Utilities (Shared): DataStorage class with static lists for Members and Trainers.
